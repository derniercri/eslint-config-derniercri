# How to use ?

Create or edit a `.eslintrc` file at project's root :

```js
{
  "extends": [
    "derniercri"
  ]
}
```

And a `.prettierrc` (`prettier` can't read `eslint` configuration) :

```js
{
  "printWidth": 100,
  "semi": false,
  "singleQuote": true,
  "trailingComma": "all"
}
```
